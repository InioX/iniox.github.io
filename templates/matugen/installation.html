Table of Contents

    Installation

    Cargo 

    cargo install matugen

    Arch

    Replace yay with your favorite AUR helper

    sudo pacman -S matugen

    NetBsd

    Using pkgin

    pkgin install matugen

    From source

    cd /usr/pkgsrc/graphics/matugen
make install

    NixOS

    Add matugen to your flake inputs

    inputs = {
  matugen = {
    url = "github:/InioX/Matugen";
    # If you need a specific version:
    ref = "refs/tags/matugen-v0.10.0";
  };
  # ...
};

    Then you can add it to your packages

    let
  system = "x86_64-linux";
in {
  environment.systemPackages = with pkgs; [    
    # ...
    inputs.matugen.packages.${system}.default
  ];
}

    NixOS/Home Manager module

    This flake also provides a NixOS/Home Manager module, which can be imported by adding this in your configuration

    {pkgs, inputs, ...}: {
  imports = [
    inputs.matugen.nixosModules.default
  ];

  # ...
}

    The module does NOT automatically symlink the files. For an example of using this module with Home Manager, see this issue comment .
    Option details can be found by reading the module.
