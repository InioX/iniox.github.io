    Default features

    Matugen gets compiled with these features by default:

    Name Default Description
    dump-json ✅ The ability to use the --dump-json flag.
    web-image ❌ The ability to use the matugen web-image subcommand.
    update-informer ❌ Will notify the user if there is an update available

    Command examples for compiling

    Compiling with specific features

    cargo build --features "<features seperated by space>"

    Compiling with all features

    cargo build --all-features
